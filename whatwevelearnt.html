What we've learnt

Week 1

Project: Jekyll Blog and Site
We've created our own blogs with Jekyll, Markdown, HTML and CSS. We've learnt about keeping our HTML dry and maintaining a separation of concerns. We hosted our site on GitHub and we learnt how to operate Git from the command line (on Terminal or on Nitrous.io).

Week 2

Project: Guardian API
We delved in Javascript and Javascript libraries (such as jQuery UI) to make our own API feed. We used Development tools to find out why sometimes our client-side code stopped working. We learnt about DOM manipulation and about responsive websites.

Week 3 & 4

Project: Twitter API
Despite just learning Javascript, we started to learn Node.js through Node School's learnyounode. We created our own web server and time server delivering the time in both ISO and Unixtime format. We learned about using Heroku to host a piece of Javascript to pull data from the Twitter API. From there, we used an Ajax function and DOM manipulation to pull the Twitter data and print tweeted images to a web page.

Week 5

Project: d3 Map
This week we continued our Javascript education and did a project using the D3 Library. The project involved parsing data from a UNODC report on cocaine prices (wholesale and retail) around the world and transforming that into a map using a colour scale linked to the price.

Week 6 & 7

Project: Bartender App
In week 6 we researched and learnt the requirements to start building our final project - an app that worked as a tinder for nearby bars. We also used this week to learn some useful methods including testing with Jasmine, lint and databasing with MongoDB. Week 7 we got stuck in to out final projects. Our app in particular required us to use the Ionic Framework for the swipe functionality, which uses angular.js. We also used this to attempt to build a sidemenu for the app.
