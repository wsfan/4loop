1. Google Chrome development tools
Originally presented by Piri on Friday, 20th June.

a. Google Chrome has in-built development tools that aid the creation and editing of webpages.
b. Access the tools by opening Google Chrome and following through to: View > Tools > Dev Tools (or on a Mac, View > Developer > Development Tools).
c. The Dev Tools bar appears at the bottom of the page providing the page's source code in a collapsable and organised format.
d. The Dev Tools bar highlights errors in coding and you can edit the source code and see changes real-time (n.b. these edits do not affect the actual webpage in any way).
